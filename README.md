# @umijs/ui-flag

A component that provides points for umi ui

## Usage

```sh
npm install @umi/ui-flag
```

in code

```tsx
import UMIUIFlag from '@umi/ui-flag';

const UserPage = ()=>{
  return (
    <>
      <ToolBar >
        // this insert button
        <UMIUIFlag />
      </ToolBar>
      // this insert table
      <UMIUIFlag />
    </>
}
```

then open umi ui  insert some block
